# 期天資訊 UI 設計分析

## 整體風格

期天資訊採用專業金融儀表板風格，具有以下特點：

### 色彩系統
- **背景色**：深藍灰色調（#1e2a3a, #2c3e50）營造專業沉穩感
- **主要色彩**：
  - 紅色（#d9534f, #c9302c）：下跌/虧損
  - 綠色（#5cb85c, #449d44）：上漲/獲利
  - 藍色（#00d8ff, #5bc0de）：中性資訊
  - 黃色（#f0ad4e）：警示/重點
- **文字色彩**：白色主文字 + 灰色次要文字

### 字體設計
- **字體大小層級分明**：
  - 標題：18-24px，粗體
  - 主要數據：16-20px，粗體
  - 次要資訊：12-14px，常規
  - 小型標籤：10-12px
- **字體家族**：使用系統字體堆疊，確保跨平台一致性
- **數字字體**：使用等寬字體或特殊數字字體，便於對齊和閱讀

### 卡片設計
- **熱力圖卡片**：
  - 大型矩形卡片，顏色深淺代表漲跌幅度
  - 顯示股票名稱、代碼、漲跌幅
  - 卡片大小與權重相關
- **資訊卡片**：
  - 圓角設計（border-radius: 8-12px）
  - 微妙陰影（box-shadow）增加層次感
  - 內邊距充足（padding: 15-20px）

### 圖示與視覺元素
- **圖示大小**：24-32px，清晰可辨
- **間距**：元素間距寬鬆（gap: 15-20px）
- **邊框**：細邊框（1px）或無邊框，依靠顏色區分

### 版面佈局
- **響應式網格**：使用 Flexbox 或 Grid
- **側邊欄導航**：固定左側，可收合
- **主內容區**：佔據剩餘空間，可滾動
- **頂部資訊欄**：顯示關鍵指標

## 適用於未實現損益 APP 的設計元素

### 1. 總損益卡片
- 採用大型卡片設計
- 背景漸層效果
- 大字體顯示金額（48-56px）
- 小圖示點綴

### 2. 股票卡片
- 左側色條標示漲跌（8px 寬）
- 卡片內分區清晰
- 使用圖示替代純文字標籤
- 增加微動畫效果（hover, transition）

### 3. 圖表優化
- 增加網格線透明度
- 使用漸層填充
- 增加互動提示
- 圖表工具列（縮放、匯出等）

### 4. 按鈕設計
- 圓角按鈕（border-radius: 8px）
- 明確的 hover 效果
- 圖示 + 文字組合
- 主要/次要按鈕層級分明

### 5. 輸入框
- 大型輸入框（padding: 15-20px）
- 清晰的 focus 狀態
- 內嵌圖示
- 自動完成提示

## 手機版優化重點

### 觸控友善
- 按鈕最小尺寸：44x44px
- 間距增加：避免誤觸
- 滑動手勢支援

### 版面調整
- 單欄佈局
- 卡片堆疊
- 底部導航列
- 可收合區域

### 字體放大
- 基礎字體：16px
- 主要數據：20-24px
- 標題：24-28px

## 建議實作的視覺改進

1. **字體系統升級**：引入 Google Fonts（Noto Sans TC + Roboto Mono）
2. **圖示系統**：使用 emoji 或 icon font
3. **動畫效果**：CSS transitions 和 transforms
4. **漸層背景**：卡片和按鈕使用微妙漸層
5. **陰影系統**：多層次陰影增加深度
6. **色彩對比**：確保 WCAG AA 標準
7. **微互動**：按鈕點擊、卡片展開等動畫
